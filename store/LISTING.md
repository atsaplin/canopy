# Chrome Web Store Listing

## Name
Canopy — Tree Style Tabs

## Short Description (132 chars max)
Tree-style tab sidebar with drag-drop hierarchy, session save/restore, AI context export, and tab staleness tracking.

## Detailed Description

Canopy organizes your browser tabs as a tree in Chrome's side panel — parent-child relationships, drag-and-drop reordering, and collapsible subtrees give you control over hundreds of tabs.

**Tree View**
• Tabs displayed as an indented tree with parent-child relationships
• Collapse/expand subtrees with one click
• Drag and drop to reparent tabs
• Tab groups shown as colored containers
• Virtualized rendering handles 500+ tabs smoothly

**Session Management**
• Save your entire tab tree as a named session (floppy disk icon)
• Load and restore previous sessions (folder icon)
• Export sessions as portable JSON files
• Import sessions from JSON files
• Browse and manage sessions in the options page

**Multi-Select**
• Click to select, Shift+click for range, Ctrl/Cmd+click to toggle
• Close, copy, or export multiple tabs at once
• "Close N tabs" shows count in context menu

**Copy & Export**
• Copy as JSON — exports selected tabs or full tree as structured JSON
• Copy context for AI — exports as markdown for ChatGPT, Claude, or any AI assistant
• Both buttons in the header, both selection-aware

**Tab Staleness**
• Tabs you haven't visited dim automatically after a configurable threshold (default 24 hours)
• Relative timestamps show how long ago you last visited each tab
• Toggle "Always show tab age" to see timestamps on all tabs

**Search**
• Instant search across all open tabs and bookmarks
• Keyboard navigable — arrow keys move through results, Enter activates
• Escape clears search and returns to tree view

**Keyboard Shortcuts**
• Alt+S toggles the side panel open/closed
• Arrow keys navigate the tree and search results
• Enter activates, Delete closes selected tab
• Shift+click for range select, Ctrl/Cmd+click to toggle

**Settings**
• Configurable indent size (compact, default, spacious)
• Stale threshold in hours (float, default 24)
• Toggle decay indicators and tab age display
• Confirm before closing a tree
• Right-click extension → Options to configure

**Onboarding**
• 5-step walkthrough on first install
• Covers tree view, sessions, export, staleness, and keyboard shortcuts

**Privacy**
• All data stays local in chrome.storage
• No external servers, no tracking, no analytics
