# Chrome Web Store Listing

## Name
Canopy — Tree Style Tabs

## Short Description (132 chars max)
Tree-style tab sidebar with drag-drop hierarchy, session save/restore, AI context export, and tab decay visualization.

## Detailed Description

Canopy organizes your browser tabs as a tree in Chrome's side panel — parent-child relationships, drag-and-drop reordering, and collapsible subtrees give you control over hundreds of tabs.

**Tree View**
• Tabs displayed as an indented tree with parent-child relationships
• Collapse/expand subtrees with one click
• Drag and drop to reparent tabs
• Tab groups shown as colored containers
• Virtualized rendering handles 500+ tabs smoothly

**Session Management**
• Save your entire tab tree as a named session
• Restore sessions (replace current tabs or add alongside)
• Export sessions as portable JSON files
• Import sessions from JSON files
• Sessions preserve the full tree structure and tab group colors

**Multi-Select**
• Click to select, Shift+click for range, Ctrl/Cmd+click to toggle
• Close, copy, or export multiple tabs at once
• "Close N tabs" shows count in context menu

**AI Context Export**
• One-click "Copy context for AI" exports your tab tree as structured markdown
• Perfect for pasting into ChatGPT, Claude, or any AI assistant
• Includes URLs, titles, and tree hierarchy

**Tab Decay**
• Tabs you haven't visited dim over time (warm → stale → decayed)
• Quickly spot tabs you've forgotten about
• Visual indicators show staleness at a glance

**Search**
• Instant regex-powered search across all tabs
• Search bookmarks too — results appear inline
• Command palette in popup (Alt+Q) for quick tab switching

**Keyboard Navigation**
• Arrow keys to navigate the tree
• Enter to activate, Delete to close
• Arrow Left/Right to collapse/expand or jump to parent

**Two UIs**
• Side panel (Alt+S): full tree view with all features
• Popup (Alt+Q): compact command palette for quick switching

**Privacy**
• All data stays local in chrome.storage
• No external servers, no tracking, no analytics
• Open source
